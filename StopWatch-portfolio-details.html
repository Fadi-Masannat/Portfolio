Stopwatch

For this project, I decided to test my limits and see if I could implement the modules and concepts from my CPRE 281 class and build my own functional stopwatch on the Altera DE2 board. The stopwatch incorporates essential functionalities such as start, stop, and reset. Additionally, it features a register file system enabling the storage of up to four laps seamlessly. While this project demanded a considerable investment of time, I am extremely pleased with the final outcome and its seamless functionality.

Components

  • 4-1 Multiplexer
  • 2-1 Multiplexer
  • 2-4 Decoder
  • Mealy & Moore State Machines
  • Register Files
  • Seven Segment Decoder
  • D-Flip Flops
  • Clock Generator
  • Clock Dividers
  • Basic NAND, AND, OR, NOR, XOR, XNOR gates

The Process



Step One:


The register file serves to store memory, utilizing a combination of 2-1 multiplexers and D-flip flops. These register files are responsible for storing time lapses, with each file possessing the capacity to store data, as depicted in the accompanying image. In total, there are four files operating in parallel, each fulfilling the same function.

The first step basically includes an up-counter that counts from 0-9. It is implemented using a combination of AND, XOR, NOT, NAND GATES, and D- Flip flops. There is a start and reset inside as input lines which gives user the ability to decide.




Step Two:


Step two also includes an up-counter that, on the other hand, counts from 0-5. It is implemented using a combination of AND, XOR, NOT, NAND GATES, and D-Flip flops. There is a start and reset inside as input lines which give the user the ability to decide when to do the needful.
